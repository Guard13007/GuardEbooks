What is it?

A simple bot using Markov chains based on a user's timeline to tweet every 15
minutes a random remix of that user. Yay.

Setup:

1. Install Python / PIP
   "sudo apt-get install python-pip" or whatever
2. Install Tweepy / PyMarkovChain
   "pip install tweepy pymarkovchain"
3. Copy secret.py.example to secret.py and modify with appropriate info.
4. "python ./bot.py"
